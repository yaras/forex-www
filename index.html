Intro.

Forex-www is a JavaScript library for technical analysis of candlestick charts. It gets array of candlestick and draw a chart. It is capable of drawing indicators (as for now only on chart, not below chart). Forex-www allows to draw custom shapes on chart like rects, lines and fibo lines.
